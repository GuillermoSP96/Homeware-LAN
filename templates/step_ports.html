Homeware configuration assistant - Ports


The first step you must acomplish is to open both 80 and 443 ports on your router.

Google will need be able to communicate with your Raspberry Pi so you need to tell your router how must route their conversations.

Open your router config web page, look for Port Forwarding and create the following rules.

  • 80 port:
    • Protocol: TCP
    • WAN start port: 80
    • WAN end port: 80
    • LAN start port: 5001
    • LAN end port: 5001
    • LAN host IP: The Raspberry Pi IP
  • 443 port:
    • Protocol: TCP
    • WAN start port: 443
    • WAN end port: 443
    • LAN start port: 443
    • LAN end port: 443
    • LAN host IP: The Raspberry Pi IP
